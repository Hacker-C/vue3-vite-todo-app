# Vue3 TodoList App

A demo app wrote by vue3 + vite:
- Vue3.2 `<script setup>` + Composition API
- Tailwindcss + Iconify

Components structure:
```bash
App
└── TodoApp
    ├── TodoForm
    └── TodoList
        └── TodoItem
```

Install
```bash
yarn
```

Develop
```bash
yarn dev
```

Build
```bash
yarn build
```
